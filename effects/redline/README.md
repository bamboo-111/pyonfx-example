# Redline

`redline.py` renders a red underline wipe with spark accents and butterfly tail launches.

## Run

```powershell
python effects\redline\redline.py --input effects\redline\test.ass --output effects\redline\output.ass --enable-multiprocessing false
```

The script loads butterfly vector frames from `butterfly/output/butterfly.10frames.0.2s.ass` relative to `redline.py`.
